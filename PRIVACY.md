# Privacy Policy for Every Second Counts

**Last updated:** January 4, 2026

## Overview

Every Second Counts is a browser extension that displays a countdown timer on your new tab page.

## Data Collection

This extension does **not** collect, store, or transmit any personal data.

## Local Storage

The extension stores only your chosen target date/time using Chrome's local storage API (`chrome.storage.sync`). This data:

- Stays on your device and synced Chrome browsers
- Is never sent to any external servers
- Is never shared with third parties
- Can be cleared by uninstalling the extension

## Permissions

- **Storage**: Used solely to save your target date preference locally

## Contact

If you have questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/vijaya22/countdown).
