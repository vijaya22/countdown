# New Tab Countdown

A minimal Chrome extension that replaces your new tab page with a countdown timer.

## Features

- Countdown timer displaying hours:minutes:seconds
- Click the target date to change it
- Clean, minimal design (off-white/black)
- Date persists across browser sessions via Chrome sync

## Installation

### From Chrome Web Store
*(Coming soon)*

### Manual Installation
1. Clone this repository
2. Open `chrome://extensions` in Chrome
3. Enable "Developer mode" (top right)
4. Click "Load unpacked" and select the project folder

## Usage

- Open a new tab to see the countdown
- Click on the target date text to open the settings modal
- Set your desired target date and time
- Click "Save"

## Privacy

This extension stores only your target date locally using Chrome's storage API. No data is collected or transmitted. See [PRIVACY.md](PRIVACY.md) for details.
